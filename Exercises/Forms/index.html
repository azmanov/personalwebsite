Forms

The <form> Element HTML forms are used to collect user input. The <form> element defines an HTML form. HTML forms contain form elements (Different types of input elements, checkboxes, radio buttons, submit buttons, and more.)

  1. Input

    2. The <input> element is the most important form element. The <input> element has many variations, depending on the type attribute. Here are the types used in this chapter:

    Type Description
    Text Defines normal text input
    Password Defines a password field
    Submit Defines a submit button (for submitting the form)
    Radio Defines a checkbox
    Checkbox Defines a submit button (for submitting the form)

    1. Input Text

      2. <input type="text"> defines a one-line input field for text input:

      Example
      <form> First name:<br>
      <input type="text" name="firstname">
      <br>
      Last name:<br>
      <input type="text" name="lastname">
      </form>
      First name:

      Last name:

    2. Input Type: password

      3. <input type="password"> defines a password field:

      Example
      <form>
      User name:<br>
      <input type="text" name="username">
      User password:<br>
      <input type="password" name="psw">
      </form>
      User name:

      User password:

    3. Input Type: submit

      4. <input type="submit"> defines a button for submitting form input to a form-handler. The form-handler is typically a server page with a script for processing input data. The form-handler is specified in the form's action attribute:

      Example
      <form action="action_page.php">
      First name:<br>
      <input type="text" name="firstname" value="Mickey">
      <br>
      Last name:<br>
      <input type="text" name="lastname" value="Mouse">
      <br><br>
      <input type="submit" value="Submit">
      </form>

      First name:

      Last name:



    4. Input Radio Button

      5. <input type="radio"> defines a radio button. Radio buttons let a user select ONE of a limited number of choices:

      Example
      <form>
      <input type="radio" name="sex" value="male" checked>Male
      <br>
      <input type="radio" name="sex" value="female">Female
      </form>

      Male
      Female

    5. Input Type: checkbox

      6. <input type="checkbox"> defines a checkbox. Checkboxes let a user select ZERO or MORE options of a limited number of choices.

      Example
      <form>
      <input type="checkbox" name="vehicle1" value="Bike"> I have a bike
      <br>
      <input type="checkbox" name="vehicle2" value="Car"> I have a car
      </form>
      <br>
      I have a bike
      I have a car

    6. Input Type: button

      7. <input type="button"> defines a button:

      Example
      <input type="button" onclick="alert('Hello World!')" value="Click Me!">



  2. The <select> Element (Drop-Down List)

    3. The <select> element defines a drop-down list:

    Example
    <select name="cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="fiat">Fiat</option>
    <option value="audi">Audi</option>
    </select>

  3. The <textarea> Element

    4. The <textarea> element defines a multi-line input field (a text area):

    Example
    <textarea name="message" rows="10" cols="30">
    The cat was playing in the garden.
    </textarea>
    <br><br>
    <textarea name="message" rows="10" cols="30">
    The cat was playing in the garden.
    </textarea>
